# Nishizumi Paints 6.5.1 Experimental

This experimental hotfix removes the old local AI roster prefill workflow that was still being triggered in the service loop.

## Fixed

- Removed the stale local AI roster prefill startup/sync path.
- Fixed the warning `Local AI roster prefill failed: name 'LocalAIRosterPrefillResult' is not defined`.
- Removed the unused prefill helper code and references.

## Kept

- Current AI random fallback behavior remains available.
- Local AI roster live updates from matching random fallback files remain available.
- AI roster sync, clone, randomize, and copy actions remain available.
