# Nishizumi Paints 7.1

This stable release promotes the tested 7.1 fixes for RandomPool, selected-driver randomization, Quick Start, and iRacing camera switching.

## Fixes

- Manual Showroom collection imports now use a collection-download filter instead of the stricter online-fallback filter, so downloadable public collection paints are not skipped just because the painter has Pro enabled or the paint includes stamped numbers.
- The Session table `Random` action now honors the Random tab source mode. If Local mode is selected, the action pulls from the Local RandomPool instead of forcing a public online random paint.
- Quick Start no longer shows the iRacing Documents folder page when automatic detection succeeds. General now keeps manual folder selection behind `Customize iRacing Documents folder`.
- Quick Start now ends on the iRacing ID for AI collections page in the normal flow, with a shorter current-ID summary.
- Double-click camera switching from the Session table now uses the Chase camera for the selected driver instead of falling back to Scenic.
