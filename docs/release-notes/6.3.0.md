# Nishizumi Paints 6.3.0

This release builds on the 6.2.0 team-paint workflow and adds more control over random public showroom fallback.

## Added

- Advanced mode now lets users choose which Trading Paints showroom sources random fallback can use.
- Supported sources are Trending, Newest, Most favorited, and Most raced (ever).
- Users can select more than one source; Nishizumi Paints will pick one of the selected sources randomly for each fallback attempt.

## Defaults

- Trending remains the default and only selected source for existing and new configurations until the user changes it.

## Notes

- The source choice applies to public showroom random fallback, selected-driver random overrides, and manual showroom RandomPool downloads.
- Numbered and PRO showroom paints are still filtered out as before.
