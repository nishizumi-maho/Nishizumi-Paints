# Nishizumi Paints 6.3.3 Draft Release Notes

## Highlights

- Showroom source defaults now start with all four sources enabled:
  - Trending
  - Newest
  - Most favorited
  - Most raced (ever)
- Existing configs that were still on the old single-source default migrate to the new all-sources default.

## Notes

- Users can still turn individual showroom sources off in Advanced mode.
- Empty source selections are normalized back to all four sources so random fallback always has a usable showroom source set.
