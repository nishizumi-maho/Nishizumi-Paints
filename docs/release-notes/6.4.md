# Nishizumi Paints 6.4 Draft Release Notes

## Highlights

- Adds a new **Showroom > Member ID** tab.
- Downloads a member's car, helmet, and suit paints by Member ID and selected car.
- Adds all-paints downloads for members into an organized `<member id>` folder.
- Adds all-paints downloads for teams into an organized `team_<id>` folder.
- Adds editable car dropdowns for Member ID and Teams so users can select mapped Trading Paints cars instead of typing IDs or directories.

## Validation

- Tested Member ID `947097` with `Acura ARX-06 GTP (TP ID 262)` and saved car, helmet, and suit files.
- Tested Member ID `947097` all-paints download and saved 29 files into per-car folders plus helmet and suit.
- Tested Team ID `404314` all-paints download and saved 17 files using batched Trading Paints manifest lookups.
