# Nishizumi Paints 6.6 Stable

This stable release adds active-driver personal paint fallback for Team sessions.

## Added

- **Use driver personal paints when team paints are missing** is enabled by default in the General tab.
- In Team sessions, missing team car, helmet, or suit assets fall back to the current in-car driver's personal paint for that same item.
- When another driver takes over the same team car, Nishizumi Paints detects the active iRacing member change, clears stale team paint files, and refreshes the team target.

## Fallback Order

1. Team paint for the item.
2. Current driver's personal paint for the item.
3. Random paint fallback when enabled.
4. iRacing default when random paints in team events are disabled.
