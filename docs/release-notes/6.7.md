# Nishizumi Paints 6.7 Experimental

This draft experimental release expands the Team-session driver fallback introduced in 6.6.

## Added

- Personal paint preloading for drivers detected in Team sessions, enabled by default.
- **Force refresh current car/team** in the Session tab.
- Separate General-tab options for using a driver's personal car, suit, and helmet when the team item is missing.
- Clearer Team Event logs for missing team items, driver personal fallback, preload cache usage, and active driver swaps.
- Session-table diagnostics for the current driver, last detected driver swap, last refresh time, and per-item source status.

## Team Fallback Order

1. Team paint for the item.
2. Preloaded current-driver personal paint when available.
3. Current-driver personal paint fetched during the session pass.
4. Random paint fallback when enabled.
5. iRacing default when random paints in Team Events are disabled.
