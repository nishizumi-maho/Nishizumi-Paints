# Nishizumi Paints 6.5.2 Stable

This stable release updates the default random-paint safety options from 6.5.1.

## Changed

- **Do not apply random paints to myself** is now checked by default for new or missing config values.
- **Do not apply random paints in team events** remains checked by default.
