# Nishizumi Paints 6.3.2 Draft Release Notes

## Highlights

- Reverted the 6.3.1 interactive **Showroom > Teams** redesign.
- Simplified **Showroom > Teams** so users only enter the required Team ID and car value.
- Kept the 6.3 Advanced mode showroom source selector for Trending, Newest, Most favorited, Most raced (ever), or a random mix of selected sources.

## Notes

- Team downloads still use the no-browser session-aware manifest flow introduced in 6.2.0.
- Optional member ID and car number context is now handled internally instead of appearing as fields in the Teams box.
