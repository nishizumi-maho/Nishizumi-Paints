# Archive

This folder contains historical Nishizumi Paints material that is kept for reference only.

## Layout

- `OLD/`
  Historical script snapshots from earlier app generations, including pre-6.x builds and embedded-browser variants.
- `SEED/`
  Older Trading Paints showroom mapping review helpers and notes.

## Intended use

This archive is useful for:

- tracing behavior that existed in earlier releases
- comparing old fallback or UI implementations
- recovering legacy ideas or migration details
- understanding the evolution from embedded-browser flows to the current no-browser/public-showroom path

## Important limitations

- Archived scripts are not maintained.
- Archived scripts are not security reviewed to the same standard as the current main script.
- Archived scripts should not be treated as supported build targets.
- The current supported application entrypoint is `Nishizumi_Paintsv6_nobrowser.py`.
